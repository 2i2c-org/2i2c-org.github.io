The [Berkeley Institute for Data Science (BIDS)](../../../collaborators/bids/index.md) has joined the [mybinder.org federation](https://mybinder.readthedocs.io/en/latest/about/federation.html), contributing a new node alongside 2i2c and [GESIS](../../../collaborators/gesis/index.md). BIDS is the birthplace of `mybinder.org`, so it's great to see them back as an active federation member.

This is a win for the resilience of mybinder.org - the BIDS node runs on a different cloud provider than the existing nodes, reducing the risk that a single provider outage takes down the whole service. 2i2c contributed expertise in extending the deployment infrastructure to this new cloud setup, and is managing some infrastructure and cloud cost payments with BIDS via their premier membership.

Read the full details on the [Jupyter blog](https://blog.jupyter.org/berkeley-institute-for-data-science-bids-joins-the-mybinder-org-federation-with-help-from-2i2c-f0f22d0b5ba5).
