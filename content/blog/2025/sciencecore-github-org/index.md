As part of the [NASA Open Science](../../../collaborators/nasa-open-science/) initiative, and in collaboration with [MetaDocencia](https://www.metadocencia.org/) (who led translation and localization of the material into Spanish), we've helped build a collection of open curriculum and tutorial material that shows researchers and educators how to analyze NASA Earthdata in the cloud — covering topics like climate risk, water resources, and OPERA Earth observation products.

You can find it here: [**github.com/sciencecore**](https://github.com/sciencecore).

As part of this work, we also operate the shared `sciencecore.opensci.2i2c.cloud` JupyterHub and an opensci BinderHub that run the hands-on exercises — both built on the open-source [JupyterHub](https://jupyter.org/hub) and [BinderHub](https://binderhub.readthedocs.io/) projects.

A few things you'll find in the GitHub organization:

- [**climaterisk**](https://sciencecore.github.io/climaterisk/) — the *Determining Climate Risks with NASA Earthdata Cloud* ScienceCore module, co-developed with MetaDocencia. ([source](https://github.com/ScienceCore/climaterisk))
- [**OPERA_Applications**](https://github.com/ScienceCore/OPERA_Applications) — interactive notebooks for working with NASA's OPERA products.
- [**TOPSTSCHOOL-module-1-water**](https://github.com/ScienceCore/TOPSTSCHOOL-module-1-water) — the TOPSTSCHOOL water module.
- [**scipy-2024-climaterisk**](https://github.com/ScienceCore/scipy-2024-climaterisk) and [**pydata-nyc-2024-climaterisk**](https://github.com/ScienceCore/pydata-nyc-2024-climaterisk) — conference tutorial versions of the climaterisk material.
- [**egis-tops-demo**](https://github.com/ScienceCore/egis-tops-demo) — an EGIS TOPS demonstration.
